## Taux de Taxes 2026

### Taux Actuels

| Taxe | Taux | Sur 100 $ |
|------|------|-----------|
| **TPS** (Fédérale) | **5%** | 5,00 $ |
| **TVQ** (Québec) | **9,975%** | 9,98 $ |
| **Total** | **14,975%** | **14,98 $** |

<InfoBox type="info">
**Important :** La TVQ est calculée sur le prix AVANT TPS, pas sur le prix incluant la TPS!
</InfoBox>

## Formules de Calcul

### Ajouter les Taxes

```
TPS = Prix × 5%
TVQ = Prix × 9,975%
Prix total = Prix + TPS + TVQ
```

**Exemple : Achat de 100 $**
- TPS : 100 $ × 5% = 5,00 $
- TVQ : 100 $ × 9,975% = 9,98 $
- **Total : 114,98 $**

### Retirer les Taxes

```
Prix avant taxes = Prix total ÷ 1,14975
TPS = Prix avant taxes × 5%
TVQ = Prix avant taxes × 9,975%
```

**Exemple : Total payé 114,98 $**
- Prix avant taxes : 114,98 $ ÷ 1,14975 = **100,00 $**
- TPS : 5,00 $
- TVQ : 9,98 $

## Produits et Services Exemptés

### Produits de Base (0% TPS/TVQ)

**Alimentation de base :**
- Pain, céréales
- Lait, œufs
- Viande, poisson
- Fruits, légumes
- Café, thé (non préparés)

**Exceptions taxables :**
- Bonbons, croustilles
- Boissons gazeuses
- Alcool
- Repas préparés (restaurants)

### Services Exemptés

**Santé :**
- Médecins, dentistes
- Médicaments sur ordonnance
- Services hospitaliers

**Éducation :**
- Frais de scolarité
- Manuels scolaires prescrits

**Services financiers :**
- Intérêts bancaires
- Assurances (la plupart)
- Hypothèques

**Logement :**
- Loyer résidentiel
- Vente maison usagée (particulier)

### Produits Taxables TPS Seulement (0% TVQ)

**Livres :**
- Livres imprimés
- Magazines, journaux

**Produits féminins :**
- Tampons, serviettes hygiéniques

## Calcul Inverse (Retirer les Taxes)

### Méthode Rapide

**Diviseur magique : 1,14975**

**Exemple : Reçu de 229,95 $**
- Prix avant taxes : 229,95 $ ÷ 1,14975 = **200,00 $**

### Méthode Détaillée

**Étape 1 : Trouver le prix avant taxes**
```
Prix avant = Prix total ÷ 1,14975
```

**Étape 2 : Calculer TPS**
```
TPS = Prix avant × 0,05
```

**Étape 3 : Calculer TVQ**
```
TVQ = Prix avant × 0,09975
```

**Exemple : 574,88 $**
- Prix avant : 574,88 $ ÷ 1,14975 = 500,00 $
- TPS : 500 $ × 5% = 25,00 $
- TVQ : 500 $ × 9,975% = 49,88 $
- **Vérification : 500 + 25 + 49,88 = 574,88 $ ✓**

### Hébergement (Hôtels)

**Taxes applicables :**
- TPS : 5%
- TVQ : 9,975%
- **Taxe d'hébergement** : 3,5% (Montréal, Québec)

- TPS et TVQ déjà dans le prix SAQ
- Taxes spécifiques sur l'alcool

### Essence

**Taxes multiples :**
- Taxe d'accise fédérale

- TPS (sur tout!)

**Environ 35-40% du prix = taxes!**

## Questions Fréquentes

### Pourquoi la TVQ n'est-elle pas 10%?

Le taux officiel est 9,975%, arrondi à 9,98% sur les reçus. C'est pour des raisons historiques de calcul.

### Les taxes sont-elles sur le pourboire?

**Non!** Le pourboire n'est jamais taxable. Calculez-le sur le montant avant ou après taxes, c'est votre choix.

### Puis-je récupérer la TPS/TVQ?

**Oui** si vous êtes travailleur autonome ou entreprise inscrite. Vous réclamez les taxes payées sur vos dépenses d'affaires.

### Les produits d'occasion sont-ils taxables?

**Dépend :** Vente entre particuliers = pas de taxes. Vente par commerçant = taxes applicables.
